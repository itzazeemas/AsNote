## ✨ Features

| Feature | Description |
|---|---|
| 🌙 Dark Mode | Eye-friendly dark theme |
| 📁 Folders | Inbox, Work, Personal, Ideas |
| 📌 Pin Notes | Important notes always on top |
| 🗄️ Archive | Hide notes without deleting |
| 🗑️ Delete | Long press to delete |
| 🎨 Color Labels | 6 colors to organize notes |
| 🔒 Capsule Notes | Lock note until a future date |
| 💣 Self-Destruct | Auto delete after 1h/6h/12h/24h/48h |
| 📊 Analytics | Note stats & folder breakdown |
| 🔍 Search | Real-time note search |
| 💾 Auto Save | Never lose your notes |
| 🔤 Font Size | Small / Medium / Large |
